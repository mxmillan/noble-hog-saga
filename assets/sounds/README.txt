NOBLE HOG SAGA — Sounds Directory
====================================

All sounds are generated procedurally via the Web Audio API — no files
are required for the game to run. However, if you want to replace any
procedural sound with a real audio file, place it here.

Expected filenames (optional overrides):

  jump.wav / jump.ogg        — Player jump sound
  collect.wav / collect.ogg  — Collectible pickup jingle
  hit.wav / hit.ogg          — Player takes damage
  defeat.wav / defeat.ogg    — Enemy stomped
  lamp_get.wav               — Magic Lamp collected
  game_over.wav              — Game over fanfare
  level_win.wav              — Level complete fanfare
  music_loop.wav             — Background ambient music loop

Supported formats: WAV, OGG, MP3 (WAV or OGG preferred for web compatibility).
Keep files small — target under 200KB each.
